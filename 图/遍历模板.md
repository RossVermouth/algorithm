## DFS

[先序遍历](https://leetcode.cn/problems/n-ary-tree-preorder-traversal/)   
[后序遍历](https://leetcode.cn/problems/n-ary-tree-postorder-traversal/)  

多叉树中不能区分一个结点是其父结点的左孩子还是右孩子，故无中序遍历。  

**递归法**
```java
class Solution {
    // 先序递归法，O(n) O(n)
    public List<Integer> preorder(Node root) {
        List<Integer> res = new ArrayList<>();
        preorder(root, res);
        return res;
    }
    // 标记并处理一个顶点，递归处理其所未被访问的邻接顶点
    private void preorder(Node root, List<Integer> list) {
        if (root == null) {
            return;
        }
        list.add(root.val);
        for (Node child: root.children) {
            preorder(child, list);
        }
    }

    // 后序递归法，O(n) O(n)
    public List<Integer> postorder(Node root) {
        List<Integer> res = new ArrayList<>();
        postorder(root, res);
        return res;
    }
    // 自左向右递归处理所有子树，再处理当前结点
    private void postorder(Node root, List<Integer> list) {
        if (root == null) {
            return;
        }
        for (Node child: root.children) {
            postorder(child, list);
        }
        list.add(root.val);
    }
}
```

**迭代法**

```java
class Solution {
    // 先序迭代法，O(n) O(n)
    // dfs用栈，为实现根-左-右，弹出一个节点时将孩子按从右到左的顺序入栈
    public List<Integer> preorder(Node root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) {
            return res;
        }
        Deque<Node> stack = new LinkedList<>();
        stack.push(root);
        while (!stack.isEmpty()) {
            Node curNode = stack.pop();
            res.add(curNode.val);
            List<Node> children = curNode.children;
            int size = children.size();
            for (int i = size - 1; i >= 0; i--) {
                stack.push(children.get(i));
            }
        }
        return res;
    }
    
    // 后序迭代法，O(n) O(n)
    // 先得到根-右-左序列，再逆置
    public List<Integer> postorder(Node root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) {
            return res;
        }
        Deque<Node> stack = new LinkedList<>();
        Deque<Node> reverse = new LinkedList<>();
        stack.push(root);
        while (!stack.isEmpty()) {
            Node curNode = stack.pop();
            reverse.push(curNode);
            for (Node child: curNode.children) {
                stack.push(child);
            }
        }
        while (!reverse.isEmpty()) {
            res.add(reverse.pop().val);
        }
        return res;
    }
}
```

## BFS

[层次遍历](https://leetcode.cn/problems/n-ary-tree-level-order-traversal/)

```java
class Solution {
    // O(n) O(n)
    public List<List<Integer>> levelOrder(Node root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) {
            return res;
        }
        List<Integer> curLevel = new ArrayList<>();
        Queue<Node> queue = new LinkedList<>();
        queue.offer(root);
        int cur = 1;
        int next = 0;
        while (!queue.isEmpty()) {
            Node curNode = queue.poll();
            curLevel.add(curNode.val);
            cur--;
            for (Node child: curNode.children) {
                queue.offer(child);
                next++;
            }
            if (cur == 0) {
                res.add(curLevel);
                curLevel = new ArrayList<>();
                cur = next;
                next = 0;
            }
        }
        return res;
    }
}
```
